#  Gurgaon Real Estate Analysis

## Project Overview

This project analyzes **19,000+ real estate property listings in Gurgaon** to identify market trends, investment opportunities, premium localities, and affordable housing options using Python.

The project answers real-world business questions and provides data-driven recommendations for buyers and investors.

---

##  Project Objectives

- Analyze Gurgaon real estate market
- Identify premium and affordable localities
- Compare Ready-to-Move vs Under Construction properties
- Analyze RERA approved properties
- Find best builders
- Perform investment analysis
- Build a Salary-Based Property Recommendation System

---

##  Dataset Information

- **Location:** Gurgaon, Haryana
- **Records:** 19,000+
- **Features:**
  - Price
  - Area
  - Locality
  - Builder
  - BHK
  - Property Type
  - Rate per Sqft
  - Status
  - RERA Approval

## Salary-Based Property Recommendation System

One of the unique features of this project is a recommendation system.

The user enters:

- Monthly Salary
- Buying Purpose
  - Self Living
  - Investment

The system estimates an affordable property budget and recommends suitable Gurgaon localities.

> **Assumption:** Estimated Affordable Budget = **5 × Annual Salary** (used only for demonstration purposes).
